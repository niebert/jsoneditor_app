About

WebApp - JSON-Editor

AppLSAC

This is a JSON editor with Load, Save and Print features running completely in the browser without submission of loaded data to remote servers. It uses Handlebars4Code to generate the output. The JSON-Editor in the repository jsoneditor_app on GitHub is used to generate tailored JSON
This is an AppLSAC. The application was create with JSONEditor4Menu. See also GitLab repository for source code of JSONEditor4Menu
